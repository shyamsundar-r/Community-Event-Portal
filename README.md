# Local Community Event Portal

A responsive web application developed using **HTML5, CSS3, JavaScript, and Bootstrap 5** that allows community members to explore events, register online, view event galleries, watch promotional videos, and find nearby events using geolocation.

---

## Project Overview

The Local Community Event Portal is designed to help residents stay informed about community activities and events. The portal provides an easy-to-use interface for event registration, event browsing, and user interaction.

---

## Features

### HTML5 Features
- Semantic HTML structure
- Navigation menu with anchor links
- Event gallery using tables and images
- Registration form with validation
- Video integration
- Geolocation support

### CSS3 Features
- Responsive design using Media Queries
- Gradient backgrounds
